Customer Shopping Behavior Analysis
📌 Project Overview

The Customer Shopping Behavior Analysis project focuses on analyzing customer purchasing patterns using transactional retail data. The objective of this project is to uncover meaningful insights related to customer demographics, spending behavior, product preferences, discounts, subscriptions, and purchasing trends.

Using Python, SQL, PostgreSQL, and Power BI, the project transforms raw shopping data into actionable business intelligence that helps businesses improve marketing strategies, customer retention, and revenue generation.

🚀 Objectives
Analyze customer shopping behavior and spending patterns
Identify high-value customer segments
Evaluate the impact of discounts and subscriptions
Discover top-performing products and categories
Generate business insights using SQL queries
Visualize insights through an interactive Power BI dashboard

🧹 Data Cleaning & Preprocessing

The data preprocessing phase was carried out using Python (Pandas & NumPy).

✔ Steps Performed
Imported and explored the dataset
Checked data structure and summary statistics
Handled missing values in review_rating
Standardized column names using snake_case
Performed feature engineering:
Created age_group
Generated purchase_frequency_days
Removed redundant columns
Loaded cleaned data into PostgreSQL for SQL analysis
